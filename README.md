# Homebrew Tap for Kanivet

This is the official [Homebrew](https://brew.sh/) tap for [Kanivet](https://kanivet.io) - Kubernetes cluster navigation and troubleshooting.

## Installation

```bash
brew tap kanivet-ai/kanivet
brew install --cask kanivet
```

## Updating

```bash
brew upgrade --cask kanivet
```

## Uninstalling

```bash
brew uninstall --cask kanivet
```

To also remove application data:

```bash
brew uninstall --cask --zap kanivet
```

## Requirements

- macOS Monterey (12.0) or later
- Apple Silicon (arm64) or Intel (x64)

## About Kanivet

Kanivet is a professional Kubernetes IDE for navigating, troubleshooting, and managing Kubernetes clusters. Features include:

- Tree-like cluster navigation
- Real-time resource monitoring
- Pod logs and metrics
- AI-powered troubleshooting
- Multi-cluster support

Learn more at [kanivet.io](https://kanivet.io)
